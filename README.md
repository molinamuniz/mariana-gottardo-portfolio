# Portfolio - Mariana Gottardo

Site de portfolio organizado para publicação no GitHub.

## Estrutura do projeto

```
Portfolio/
  index.html
  assets/
    css/
      style.css
    js/
      main.js
    images/
      projeto-c-h/
      casa-ra/
      quarto-arthur/
    docs/
      *.pdf
```

## Publicar no GitHub

1. Crie um repositório no GitHub.
2. Envie todo o conteúdo desta pasta `Portfolio` para a branch `main`.
3. No GitHub, abra `Settings > Pages`.
4. Em `Build and deployment`, selecione:
   - `Source`: Deploy from a branch
   - `Branch`: `main`
   - `Folder`: `/ (root)`
5. Salve e aguarde alguns minutos.
6. O site ficará disponível no link informado na seção `Pages`.

## Atualização de conteúdo

- Edite estrutura e conteúdo em `index.html`.
- Edite estilos em `assets/css/style.css`.
- Edite interações em `assets/js/main.js`.
- Troque imagens em `assets/images/...`.
- Troque PDFs em `assets/docs/`.
